EBS scale (matches _layouts/intervention.html ebsMagnitudeLabel): slight |EBS| < 1 modest 1 ≤ |EBS| < 3 meaningful 3 ≤ |EBS| < 5 strong 5 ≤ |EBS| < 7 transformative |EBS| ≥ 7 Each EBS point ≈ 2× the impact of the previous, so the gap between 'slight' and 'transformative' is roughly 128× in absolute benefit. {% endcomment %}

    Magnitude labels come from the Expected Benefit Score (EBS), which combines magnitude of benefit, success rate, adherence, and confidence on a logarithmic scale. Each EBS point ≈ 2× the previous step's impact.
